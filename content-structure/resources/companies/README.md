Defense and aerospace contractor using Pharo for:
- UI/UX prototyping for defense systems
- Support Wizard desktop applications
- Human-machine interfaces

### Barger-Levrault (France)
Software company using Pharo for:

- Legacy system modernization

### QqDataFruits (France)
Data integration platform built with Pharo:
- Data consolidation and reporting
- Automated functional data processes

### Apptive (Germany)

- ApptiveFleet - Vehicle management
