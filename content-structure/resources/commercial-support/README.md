# Commercial support

## Purpose

Information about professional support options for companies using Pharo, including consortium and consulting services.

## Where this can be used

- **Community hub** - Support directory
- **Main Pharo site** - Section for support
- **Contribute hub** - Coporate contribution benefits
- **Companies site** - Service provider information

## Consortium membership

The Pharo Consortium includes limited commercial support with membership.

**What's included:**
- Core engineer time
- Privileged access to core development team
- Training special prices
- Priority isses handling

## Consulting services

If you are looking for Pharo consultants:

**Contant:** The Pharo consortium can connet you with qualified consultants.

**Services available:**
-Traning and onboarding
- Architecture review
- Custom development
- Migration assistance

## Comercial support infrastructure

For commercial support infrastructure specifically for Pharo:

**Contat:** Pharo board

**What this covers:**
- Enterprise-level support agreements
- Infrastructure hosting support
- Custom SLAs

## For service providers

Companies offering Pharo services can:
- Join the Consortium
- List in consulting directory
- Present at consortium mettings
