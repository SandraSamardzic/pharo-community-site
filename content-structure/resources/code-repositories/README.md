# Code repositories

## Purpose

Directory of code repositories where Pharo projects are hosted, both current and legacy.

## Where this can be used

- **Community hub** - Repository links
- **Main Pharo site** - Section where you need to acess code
- **Contribute hub** - Where to submit code
- **GSoC site** - Project repositories

## Primary repository

### GubHub
Most Pharo projects are now saved on GitHub

**Important:** Tag your projects with the 'pharo' tag so other can find them.

**Links:**
- [Search Pharo projects on GitHub](https://github.com/search?q=pharo)
- [Pharo organization on GitHub](https://github.com/pharo-project)
