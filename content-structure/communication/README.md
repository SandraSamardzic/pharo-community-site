# Communication channels

## Purpose

Centeral hub for all Pharo communication platforms where users and developers discuss, ask questions, and share knowledge.

## Where these links can be used

- **Comuunity hub** - Primary communication directory
- **Main Pharo site** - Section where to seek help or other feedback
- **Newcomers site** - Where beginners can ask questions
- **Documentation site** - Links for further help

## Channels

### Discord team
Real-time chat for Pharo users and developers. Most active and immediate support channel.

**Link:** https://discord.gg/QewZMZa

### Mailing lists
Primary communication tool for development discussions and user questions.

**Main lists:**
- **pharo-users** - Friendly place for any question about Pharo. [Archive] (https://lists.pharo.org/pipermail/pharo-users/)
- **pharo-dev** - Main communication for Pharo development. [Archive](https://lists.pharo.org/pipermail/pharo-dev/)

- **Subscribe:** https://lists.pharo.org

- ### Pharo newsletter
- Monthly overview of projects happening in Pharo.

- **Status:** Currently on hold
- **Archive:** Available

### StackOverflow
Q&A platform for technical questions

**Usage:** Tag questiong with 'pharo' to ensure they are seen.
**Note:** Community prefers mailing list or Discord, but StackOverflow is available.

**Link:** https://stackoverflow.com/questions/tagged/pharo
