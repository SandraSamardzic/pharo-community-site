# Students & newcomers

## Purpose

Resources for students, newcomers, and anyone their journey with Pharo.

## Where this can be used

- **Community Hub** - Student resources directory
- **Newcomers site** - Primary learning destination
- **GSoC site** - Student program information
- **Main Pharo site** - Section for getting started

## Learning resources

### Newcomers site
Complete resources for people new to Pharo.

**Includes:**
- Getting started guides
- Tutorials and exercises
- Learning paths
- Community connections

**Link:** https://newcomers.pharo.org

### Documentation site
Technical documentation and reference materials.

**Link:** https://documentation.pharo.org

## Programs

### Google Summer of Code (GSoC)
Annual program where students contribute to open source projects.

**Opportunities:**
- Paid summer internship
- Work with Pharo mentors
- Build real projects
- Join the community

**Link:** https://gsoc.pharo.org

### Summer school
Educational programs for students interested in Pharo and Smalltalk.

**Check the newcomers site for announcments and schedules.**
