# Pharo community hub

This site will be a part of the Pharo Web Ecosystem using a modular approach, centeralizing all community and help resources, from Discord discussions to research groups and commercial support.

**Potential URL:** https://community.pharo,org

## Planned modules

- **pharo-web-core** - Base styles and utilities
- **pharo-web-navigation** - Community-specific navigation
- **pharo-web-communication** - Chat, forums, mailing lists, social media
- **pharo-web-ecosystem** - Code repositories, companies, research groups
- **pharo-web-support** - Commercial support nd consulting

## Content strucutre

The content will be organized in the 'content-structure/' folder with the following strucutre:

### Communication channels

- **Discord** - Real-time chat for users and developers
- **Mailing lists** - pharo-users, pharo-dev, and newsletter
- **Q&A platforms** - StakOverflow (with pharo tag)
- **Social media** - Mastodon, Bluesky, Facebook, LinkedIn
- **Blogs** - Pharo Weekly, Astares, teaching blog

### Ecosystem & resources

- **Code repositories** - GitHub (primary), SmalltalkHub, SS3
- **Companies** - Businesses using Pharo worldwide
- **Research groups** - Universities and labs using Pharo
- **Support** - Commercial support and consortium consulting

### Related initiatives

- **Newcomers** - Link to https://newcomers.pharo.org
- **GSoC** - Student opportunities
- **Summer school** - Education programs

## Future integrations

This community hub will be reusable across:

- **Main Pharo site** - "Community" navigation section
- **Newcomeers site** - Where to get help as a beginner
- **GSoC site** - Mentor and student communication channels
- **Contribute hub** - How to join communiy discussions
- **Success stories site** - Company and research group spotlights
